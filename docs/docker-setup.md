### Запуск контейнера:


Через командную строку (CMD):



docker run -d -p 5000:8080 --name lb ^

&#x20; -e "Settings\_\_Backends\_\_0\_\_Host=192.168.1.10" ^

&#x20; -e "Settings\_\_Backends\_\_0\_\_Port=5101" ^

&#x20; -e "Settings\_\_Backends\_\_1\_\_Host=192.168.1.11" ^

&#x20; -e "Settings\_\_Backends\_\_1\_\_Port=5102" ^

&#x20; loadbalancer:v1



\-d - запустить в фоне

\-p 5000:8080 - порт 5000 на компьютере → порт 8080 в контейнере

\--name lb - назвать контейнер "lb"

loadbalancer:v1 - имя образа



#### Как добавлять переменные окружения:


-e "Путь\_\_К\_\_0\_\_Настройке=значение"

(два подчеркивания это вложенность)

(число нужно для индекса в массиве)



Затем открыть браузер http://localhost:5000
