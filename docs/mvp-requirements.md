# MVP — Требования к L7 балансировщику (HTTP/HTTPS) 

Reverse Proxy на с#, который принимает HTTP/HTTPS запросы и распределяет их между backend-серверами.

## Описание работы:
- Принимает входящие HTTP/HTTPS запросы
- Перенаправляет их на backend-сервера c помощью алгоритма балансировки запросов Weighted Round Robin
- Возвращает клиенту ответ без изменений

## Список настраиваемых backend-серверов хранится в конфигурационном файле 
- с адресом
- портом
- и весом

## В логирование входят:
- входящий запрос
- выбранный сервер
- ошибки
